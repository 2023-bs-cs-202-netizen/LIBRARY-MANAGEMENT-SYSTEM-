# LIBRARY-MANAGEMENT-SYSTEM-
Library Management System is a 5th semester C++ project designed to manage books, users, issuing, and returning efficiently. It uses object-oriented programming concepts and a menu-driven interface. Developed by Laiba Anjum (196), Summaya Muzzamal (202), Malaika (226), and Raiba Rasheed (246).

Overview of Library management system(LMS)

Definition

A Library Management System is a software application used to manage, organize, and automate library operations such as book issuing, returning, cataloging, and member management.

Objectives

* Automate manual library work
* Maintain accurate book records
* Reduce human errors
* Save time and improve efficiency
* Provide quick search and reporting

Main Users

* Admin / Librarian
* Students / Members/Faculty

Core Modules / Components

1. User Management

* Add, update, delete members
* Maintain student/staff records
* Login & authentication

 2. Book Management

* Add new books
* Update book details
* Remove damaged/lost books
* Manage book categories

 3. Issue & Return Management

* Issue books to members
* Return books
* Calculate fines for late returns
* Track issued books

 4. Search Module

* Search by title
* Search by author
* Search by ISBN
* Search by category

 5. Fine Management

* Automatic fine calculation
* Fine payment record

 6. Reports Module

* Issued books report
* Returned books report
* Overdue books report
* Member activity report

Functional Requirements

* User login system
* Add/edit/delete books
* Issue and return books
* Search functionality
* Fine calculation

Non-Functional Requirements

* Easy to use interface
* Fast response time
* Secure data handling
* Reliable system
* Scalable for future use

Technologies Used (Example)

* Used Language: C++
* Tools:Visual Studio,DEV C++

Real-Life Example

In a college library, instead of maintaining registers manually, the librarian uses the Library Management System to issue books, check availability, and generate reports instantly.
Advantages

✔ Time saving
✔ Accurate data
✔ Easy book tracking
✔ Reduced paperwork
✔ Better management
Conclusion

A Library Management System improves library efficiency by automating routine tasks and maintaining accurate records, making it essential for modern educational institutions.
